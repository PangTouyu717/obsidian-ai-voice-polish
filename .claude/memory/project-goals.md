## 核心工作流

```
录音（麦克风） → 语音转文字（STT） → AI 润色 → 自动输出到光标位置/当前笔记
```

## 目标

上架 Obsidian 社区插件市场，手机端（iOS/Android）可以直接在 Obsidian App 中搜索安装使用。

## 功能需求

1. **语音录制**：按下快捷键开始录音，松开/再次按下结束
2. **语音转文字**：调用 API 将音频转为文本
3. **AI 润色**：对转写文本进行润色（去口头禅、修正错字、整理语序等）
4. **自动输出**：润色后的文本直接插入到当前光标位置或新建笔记
5. **多 API 支持**：DeepSeek / 豆包（火山引擎）/ 千问

## 上架要求

- 遵循 Obsidian 插件开发规范
- 代码开源、清晰
- 隐私合规（API Key 不能明文硬编码）
- 移动端兼容（UI 适配、音频 API 兼容）
